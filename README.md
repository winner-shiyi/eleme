## 演示

<a href="http://www.iswn.me/eleme-vue2/#/goods" target="_blank">在线演示</a>

## 安装步骤

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
